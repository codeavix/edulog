# edulog
A simple **Student Record and Grade Management System** developed in C as a first-semester engineering project. The application allows management of student records, marks, GPA/CGPA calculation, grade generation, report cards, and basic class performance analytics through a menu-driven console interface. 
Sure! Since this looks like a **first-semester C programming project** for a student management/result processing system, I'd make the README sound realistic, humble, and human-written rather than like a corporate software project.

## Features

### Student Management

* Add new student records
* Display all students
* Search for a student
* Delete student records
* Store personal information such as:

  * Roll Number
  * Name
  * Date of Birth
  * Gender
  * Batch

### Marks Management

* Enter and modify marks
* Maintain marks for multiple subjects
* Track assessment scores for each student

### Grade Processing

* Calculate subject-wise grades
* Calculate GPA and CGPA
* Calculate overall percentage
* Identify backlogs/failures
* Generate student report cards

### Student Portal

Students can:

* View personal details
* View academic performance
* Access their report card

### Academic Analytics

* Class statistics
* Subject toppers
* Overall toppers list
* Pass list
* Fail list
* Mean, median, and mode calculations
